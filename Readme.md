# 작은옷장 
ARKit의 bodyTracking을 사용한 Unity기반 AR 가상피팅서비스 

## Requirements
- Unity 2021.3.8f1
- ARFoundation/ARKit 3.1.0
- XCode 14.1
- iPad 15.0 / iPhone 12.0
